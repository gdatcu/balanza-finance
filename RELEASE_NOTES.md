# Balanza Finance v1.9.0 Release Notes

## ⚡ New Features

### 📅 Group Transactions by Date & Daily Net Totals Header
- Upgraded Dashboard transaction list to group transactions by calendar date.
- Added styled Date Group Headers displaying:
  - **Left**: Localized Date (e.g. `Today` / `Astăzi`, `Yesterday` / `Ieri`, or localized `July 24, 2026`).
  - **Right**: Formatted Daily Net Total (`+` Green for positive net, `-` Red/Coral for negative net).
  - Multi-currency formatted daily totals respecting active currency settings.
- Group headers feature subtle container styling with background `#1E293B` and rounded borders for clear visual separation.
